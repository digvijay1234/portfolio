Project

Mobile Webpage for Ferrari SF23 Formula 1 Car.

Objective

To create an engaging mobile experience showcasing the Scuderia Ferrari SF23 for 2023, highlighting its features and performance specs.

Tools Used

Design Approach

User Interface:

  • Clean, minimalistic design with Ferrari branding.
  • High-quality, full-width image of the SF23 car.

Navigation:

  • Simple, intuitive menu for easy access to additional information.
  • Language selection for a global audience.

Content:

  • Key information about the SF23, including technical specs and performance highlights.
  • Visual emphasis on the car with minimal text to maintain focus on imagery.

Features

  • High-Resolution Imagery: Centralized large image of the SF23 for visual impact.
  • Interactive Elements: Menu and language selection to enhance user experience.
  • Responsive Design: Optimized for mobile devices ensuring accessibility and usability.

Process

3D Asset Integration:

  • 3D models of the SF23 were sourced from Sketchfab and integrated using Spline for a realistic representation.
  • High-quality, full-width image of the SF23 car.

Animation Creation:

  • Anima app was used to animate the car, creating dynamic visual interest and interactive elements.

Prototyping and Testing:

  • Figma and anima app was utilized to prototype the mobile webpage, ensuring an intuitive user experience.

Results

Enhanced Engagement:

  • The visually captivating design attracted and retained user interest.

Improved User Experience:

  • Simple navigation and clean layout made information easily accessible.

Brand Reinforcement:

  • Strong Ferrari branding throughout the webpage.

Conclusion
